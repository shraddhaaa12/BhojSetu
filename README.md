# BhojSetu

BhojSetu is a full-stack food donation platform that connects food donors, restaurants, households, and event organizers with NGOs to reduce food wastage and support SDG 2: Zero Hunger.

## Features

- Food Donation Portal
- NGO Registration System
- Unique Donation Tracking ID
- Dynamic Dashboard
- AI-assisted NGO Recommendation
- Community Impact Tracking
- Food Safety Declaration

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON Database

## SDG Alignment

SDG 2 – Zero Hunger
